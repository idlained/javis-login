# Javis Login Challenge

## 📌 Deskripsi
Aplikasi login sederhana dengan autentikasi JWT menggunakan Node.js dan React.js.

---

## 🚀 Features
- Login (Email / Username)
- Validasi form
- JWT Authentication (HttpOnly Cookie)
- Protected Dashboard
- Logout
- Show/Hide Password
- Responsive UI
- Dark Mode
- Loading animation
- Rate limit login

---

## 🛠 Tech Stack
### Frontend
- React.js
- TailwindCSS
- Axios

### Backend
- Node.js (Express)
- MySQL
- JWT
- Bcrypt
